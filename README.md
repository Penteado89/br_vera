# BR-VERA Annotation Codebook

This repository hosts the public annotation codebook for **BR-VERA**  
(Brazilian Violent Extremism Risk Analysis corpus).

The BR-VERA annotation effort is part of the **BR-ECHO** project, focused on
auditable, evidence-grounded resources for violent extremism risk analysis
in Brazilian Portuguese.
